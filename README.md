## Setup
`You can use either npm or yarn`
```bash
cd web
npm install / yarn install
ng serve --o
```

### How to get start?
##### To create components
`Make sure you're in the angular project root`
```bash
ng g c <path/to/file>
```

##### To create services
`Make sure you're in the angular project root`
```bash
ng g s <path/to/file>
```

##### To build for production
`Make sure you're in the angular project root`
```bash
ng build --prod
```
